VPN - Virtual Private Network

To connect to the internal network using the VPN client, you will need to download and configure it on your computer.
You will also need a user name and password.

1. Download VPN client

  • Click here to download the Windows installer (25,4 MB - 32 and 64-bit)

2. Install VPN client

  • Double click the installation file you have downloaded above: CheckPointVPN.msi.
  • Confirm the execution
  • Click Next
  • Ensure Endpoint Security VPN is selected and click Next
  • Select I accept the terms in the license agreement and click Next
  • Click Install. You will be prompted to enter an Administrator credential
  • Click Finish. Reboot if necessary

3. Configure the VPN connection

  • Click Yes to configure a new site
  • Click Next
  • Enter the server name vpn.embraco.com and click Next
  • Click Trust and Continue
  • Select Username and Password and click Next
  • Click Finish

4. VPN - Connection instructions

  • Double click the VPN icon on the taskbar
  • Enter your username and password and click the Connect button
  • Enter the server name vpn.embraco.com and click Next
  • Wait the VPN connection to be established
  • When it is connected, the VPN taskbar icon changes to indicate it
  • To end the VPN connection, double click the VPN taskbar icon and choose Disconnect
